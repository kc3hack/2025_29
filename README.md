## プロダクト説明

このプロジェクトは、"RaspberryPi 3" を使用します。
冷蔵庫の開閉時に 庫内撮影・画像解析 を行うことで、開閉時に取り出した(庫内から消えた)食材のカロリーを記録するシステムです。
バックエンドにはHonoを使用しています。
